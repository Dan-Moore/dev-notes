### Creating a Partition table
Run parted with the command `mklabel gpt` to make a [GUID Partition Table](https://wiki.archlinux.org/title/GUID_Partition_Table) or `mklabel msdos` for a [Master Boot Record](https://wiki.archlinux.org/title/Master_Boot_Record)

```bash
# SSDs (alpha & delta)
parted -s /dev/sda mklabel gpt
parted -s /dev/ssd mklabel gpt

# HDDs (beta & cappa)
parted -s /dev/sdb mklabel gpt
parted -s /dev/sbc mklabel gpt
```
    
Creating the partition.

```bash
parted -s /dev/sda mkpart primary ext4 0% 100%
parted -s /dev/sdb mkpart primary ext4 1 2500
```
